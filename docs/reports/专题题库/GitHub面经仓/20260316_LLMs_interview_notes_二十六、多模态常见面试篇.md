## 专题总结

## 本专题考什么
- **多模态基础范式**：考察对图文对齐（如对比学习）、跨模态融合（如交叉注意力）以及多模态生成（如自回归、扩散模型）底层逻辑的理解。
- **经典与前沿模型演进**：评估候选人是否清晰掌握从 CLIP（双塔）到 BLIP 系列（融合编码器），再到目前主流的基于 LLM 的多模态大模型（如 LLaVA、BLIP-2）的发展脉络。
- **架构设计与工程权衡**：重点考察跨模态桥接模块（Adapter/Q-Former/Projector）的设计原理，以及在算力受限下如何高效训练（如冻结大模型权重、两阶段训练）。

## 答题主线
- **宏观演进线**：答题时应具备“时间线”意识。从“特征对齐”（CLIP）讲到“深度融合”（ALBEF/BLIP），再讲到“大语言模型赋能”（BLIP-2/LLaVA），展现全局视野。
- **微观技术线**：拆解模型时，固定按照 **“图像编码器 (ViT) + 跨模态桥接模块 (Q-Former/MLP) + 文本解码器 (LLM)”** 的三段式结构进行分析。
- **核心亮点线**：强调模型的训练目标（如 ITC、ITM、ITG）和工程优势（如参数高效微调、冻结预训练权重降低算力成本）。

## 代表题答法

### 问题1：最近关注的论文，多模态视觉大模型(CLIP, DALLE)？
**答题策略**：不要只报菜名，要以 CLIP 和 DALL-E 为基石，引出多模态的“理解”与“生成”两大分支，并结合最新进展展现技术敏感度。

**核心答法**：
1. **CLIP（理解与对齐的基石）**：
   - **核心机制**：采用双塔架构（Image Encoder + Text Encoder），通过海量图文对进行**对比学习（InfoNCE Loss）**，将图像和文本映射到同一特征空间。
   - **历史地位**：打破了传统分类模型的闭集限制，实现了强大的 Zero-shot 能力，成为后续几乎所有多模态模型（包括 DALL-E 2、Stable Diffusion）的标配视觉/文本特征提取器或对齐指导。
2. **DALL-E 系列（生成的代表）**：
   - **演进**：DALL-E 1 使用 dVAE 将图像离散化并用 Transformer 自回归生成；DALL-E 2 引入了 Diffusion 模型，并利用 CLIP 的文本嵌入（Text Embedding）通过 Prior 网络生成图像嵌入（Image Embedding），再解码为图像。
3. **前沿延伸（加分项）**：
   - 面试时可补充最近关注的基于 LLM 的多模态模型（如 LLaVA 的简单线性映射，或 Sora 在视频生成中的 DiT 架构），强调当前趋势已从“专用多模态网络”转向“统一的 Transformer 架构”和“大语言模型作为大脑”。

### 问题2：blip2的架构，优势和之前多模态模型的区别？
**答题策略**：直击 BLIP-2 的核心创新——**Q-Former** 和 **两阶段训练法**，对比前代模型突出其“四两拨千斤”的算力优势。

**核心答法**：
1. **BLIP-2 架构拆解**：
   - 核心由三部分组成：冻结的图像编码器（ViT）、冻结的大语言模型（LLM，如 OPT/FlanT5）、以及中间可训练的 **Q-Former（Querying Transformer）**。
   - Q-Former 包含两组 Transformer 子模块：Image Transformer（与视觉特征交互）和 Text Transformer（处理文本）。它通过一组可学习的 Learnable Queries 提取与文本最相关的视觉特征。
2. **两阶段预训练机制**：
   - **阶段一（视觉-语言表征学习）**：冻结 ViT，训练 Q-Former。优化目标包括图文对比学习（ITC）、图文匹配（ITM）和基于图像的文本生成（ITG）。
   - **阶段二（视觉到语言生成学习）**：冻结 ViT 和 LLM，将 Q-Former 提取的视觉特征通过全连接层投影到 LLM 的输入空间，作为 Soft Prompt 引导 LLM 生成文本。
3. **核心优势**：
   - **极高的计算效率**：由于冻结了参数量最大的 ViT 和 LLM，只需训练轻量级的 Q-Former（约 100M 参数），大幅降低了训练成本。
   - **避免灾难性遗忘**：冻结 LLM 保证了模型不会在多模态微调中丧失原有的强大文本推理和生成能力。
4. **与之前模型的区别**：
   - **对比 CLIP**：CLIP 是浅层交互（双塔点乘），缺乏文本生成能力；BLIP-2 接入了 LLM，具备强大的 VQA 和图像描述生成能力。
   - **对比 BLIP-1/ALBEF**：早期模型通常需要端到端微调整个网络，计算代价大；且 BLIP-2 引入的 Q-Former 充当了**信息瓶颈（Information Bottleneck）**，能过滤掉无关的视觉背景噪声，只喂给 LLM 最核心的视觉信息，这是前代模型不具备的。
