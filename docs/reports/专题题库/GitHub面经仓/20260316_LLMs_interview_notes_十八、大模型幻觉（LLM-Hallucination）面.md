## 本专题考什么
- **核心概念理解**：考察候选人对“幻觉”定义的准确把握，能否区分不同类型的幻觉（如内在幻觉与外在幻觉、事实性幻觉与逻辑性幻觉）。
- **底层机制剖析**：考察对大模型（自回归语言模型）本质的理解，能否从数据、训练目标（Next-token prediction）、解码机制等维度深度剖析幻觉成因。
- **工程解决能力**：考察候选人在实际业务中解决幻觉问题的工具箱储备，涵盖从数据清洗、模型微调（SFT/RLHF）到推理端优化（RAG、Prompt工程、解码策略）的全链路缓解方案。

## 答题主线
- **先定性与分类**：一句话点透幻觉本质（“一本正经地胡说八道”），并按表现形式进行专业分类，展现基础扎实度。
- **分阶段找原因**：按照大模型的生命周期（预训练数据 -> 训练机制 -> 对齐阶段 -> 推理生成）逐层拆解成因，体现系统性思维。
- **全链路给方案**：对应成因提出缓解策略。面试中应重点突出**低成本/高收益的工程方案**（如 RAG、CoT、拒绝回答训练），辅以**模型层面的前沿解法**（如 DoLa解码、事实性RLHF），展现技术视野与实战经验。

## 代表题答法

### 问题1：什么是大模型幻觉（LLM Hallucination）？
**答题要点：**
大模型幻觉是指模型生成了**语法流畅、看似合理，但实际上与客观事实不符、与输入上下文矛盾或毫无根据**的内容。
在面试中，建议将其细分为以下几类以体现专业度：
1. **事实性幻觉（Factual Hallucination）**：生成的内容与现实世界的客观事实相悖（例如说“林黛玉倒拔垂杨柳”）。
2. **内在幻觉（Intrinsic Hallucination）**：生成的内容与用户提供的Prompt或上下文存在逻辑冲突（例如摘要任务中捏造了原文没有的数据）。
3. **外在幻觉（Extrinsic Hallucination）**：生成的内容虽然不一定错，但无法从输入源中找到依据，属于模型无中生有的过度发散。

### 问题2：为什么LLM会产生幻觉？
**答题要点：**
幻觉不是Bug，而是自回归模型当前架构的“Feature”。成因需从三个维度拆解：
1. **数据层面（源头污染与知识盲区）**：
   - 预训练数据中本身包含大量互联网噪声、过时信息或矛盾观点。
   - 知识长尾效应：对于训练数据中出现频次低的罕见知识，模型容易靠“猜”来补全。
2. **训练机制层面（目标错位）**：
   - **Next-token prediction 的局限**：自回归模型的本质是拟合词汇的概率分布，它倾向于生成“统计上最连贯”的句子，而非“事实上最准确”的句子。
   - **暴露偏差（Exposure Bias）**：训练时采用Teacher Forcing（基于真实前缀预测），推理时基于自身生成的Token预测，一旦前面生成了一点偏差，后面就会发生“雪崩式”的幻觉（滚雪球效应）。
3. **对齐与推理层面（谄媚与随机性）**：
   - **RLHF的副作用**：模型为了获得更高的人类偏好奖励，可能会产生“谄媚（Sycophancy）”现象，即迎合用户Prompt中的错误预设。
   - **解码策略**：较高的 Temperature 或 Top-p 采样会引入随机性，增加偏离事实的概率。

### 问题3：如何缓解大模型幻觉？
**答题要点：**
缓解幻觉需要打组合拳，按实施成本从低到高、从推理端到训练端依次展开：
1. **推理端与外部增强（见效最快，工程首选）**：
   - **RAG（检索增强生成）**：引入外部知识库，将生成任务转化为“基于检索内容的阅读理解”，极大降低事实性幻觉。
   - **Prompt Engineering**：使用 CoT（思维链）强制模型暴露推理过程；加入“如果你不知道，请回答不知道”的系统提示词，限制模型发散。
   - **解码策略优化**：降低 Temperature；使用前沿的对比解码技术（如 DoLa，通过对比不同网络层的Logits来放大事实性知识的权重）。
2. **微调阶段（SFT / RLHF）**：
   - **拒绝回答训练（Refusal Training）**：在SFT数据中刻意构造超出模型知识边界的边界Case，教会模型输出“我不知道”，校准模型的置信度。
   - **事实性对齐**：在RLHF阶段，专门训练针对“事实准确性”的Reward Model，惩罚幻觉输出（如使用 DPO 算法针对幻觉数据进行负向优化）。
3. **预训练与数据阶段（成本最高，治本之策）**：
   - 提升预训练数据的质量，加强去重（Deduplication）和事实性清洗。
   - 知识图谱增强：在预训练阶段引入结构化知识，提升模型对实体关系的记忆准确度。
