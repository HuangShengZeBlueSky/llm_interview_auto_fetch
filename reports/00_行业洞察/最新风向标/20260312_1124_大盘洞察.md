## 🔥 二、 高频考点红榜

以下是本周出场率最高、且具有极高区分度的硬核考点。它们之所以火爆，完全是由当前的工业界业务痛点决定的：

### 🥇 Top 1: KV Cache 与 GQA (Grouped-Query Attention)
*   **业务背景解释**：大模型推理的本质瓶颈根本不是算力（FLOPs），而是**显存带宽（Memory Wall）**。随着上下文窗口（Context Length）动辄逼近 128K，标准的 MHA 架构会导致 KV Cache 撑爆显存，Batch Size 只能开到 1，机器疯狂空转。GQA 作为 MHA 和 MQA 的完美折中，在几乎不损失模型表达能力的前提下，将显存占用降低了数倍。**它是目前 Qwen2、Llama3 等所有主流大模型的标配，不懂 GQA 等于不懂现代大模型架构。**

### 🥈 Top 2: FlashAttention (IO-Awareness 与 Online Softmax)
*   **业务背景解释**：天下苦 $O(N^2)$ 复杂度久矣。但面试官考 FA，绝不是让你背诵时间复杂度，而是考你**“硬件感知（Hardware-aware）”**。业务痛点在于 GPU 的 HBM（主存）读写太慢，而 SRAM（片上内存）太小。FA 通过 Tiling（分块）和 Online Softmax 数学等价变换，把中间庞大的 Attention 矩阵死死按在 SRAM 里计算，避免了 HBM 的反复读写。这是大模型长文本训练得以实现的**物理基石**。

### 🥉 Top 3: SFT 的局限性与 DPO (直接偏好优化)
*   **业务背景解释**：业务上发现，光靠 SFT（行为克隆/Next Token Prediction），模型只会“模仿”语气，却依然会产生严重幻觉，且无法区分“普通回答”和“极佳回答”。传统的 RLHF (PPO) 训练链路太长、极度不稳定且需要额外训练 Reward Model，成本极高。DPO 通过巧妙的数学推导，直接用交叉熵在语言模型上做对比学习（Chosen vs Rejected），极其轻量且效果拔群，已成为当前大厂做 Alignment（价值观与逻辑对齐）的绝对主流。

### 🏅 Top 4: GraphRAG 与 垂直领域 RAG 架构
*   **业务背景解释**：传统的“切块+向量检索”RAG 在企业级落地中已经被证明是“玩具”，它只能回答细节问题，遇到“请总结这份财报的核心风险”这种宏观多跳问题直接抓瞎。GraphRAG 通过引入大模型构建知识图谱和社区摘要（Community Summaries），结合 Map-Reduce 架构，真正解决了长文本全局理解的痛点。

## ⚔️ 三、 能力代差分析：面试官的“连环追问”杀招

在筛选简历和面试时，普通候选人和高级候选人的分水岭，往往体现在对**“边界条件”**和**“工程细节”**的掌控力上。以下是本周面经中，面试官最爱用的“降维打击”式追问：

### 场景 1：考察底层逻辑的严密性
*   **初级回答**：知道 RoPE 是旋转位置编码，能兼顾绝对和相对位置。
*   **高级追问**：*“引入 RoPE 后，KV Cache 里存的 K 是编码前的还是编码后的？为什么？”*
*   **代差体现**：普通人死记硬背概念；高级候选人能在脑海中推演计算图——因为 RoPE 是绝对位置的相对表达，当前 Token 的 Q 乘以历史的 K 时，K 必须已经注入了绝对位置信息，所以**必然缓存的是编码后的 K**。

### 场景 2：考察对训练稳定性的直觉
*   **初级回答**：知道 LoRA 是低秩微调，冻结主干，训练 A 和 B 两个小矩阵。
*   **高级追问**：*“LoRA 的 A 和 B 矩阵能都用高斯初始化吗？或者互换初始化方式？”*
*   **代差体现**：高级候选人深知**“旁路网络不能破坏预训练初始分布”**的铁律。必须保证 B 全零初始化，使得初始状态下 $BA=0$，否则一开始巨大的随机噪声会让 Loss 直接爆炸。

### 场景 3：考察工程落地的真实体感
*   **初级回答**：知道 DPO 需要正负样本（Chosen / Rejected）。
*   **高级追问**：*“怎么自动构造难负样本（Hard Rejected）？如果模型在 Function Call 时虚构了不存在的 API 参数（幻觉），工程上和算法上分别怎么兜底？”*
*   **代差体现**：没有真实踩过坑的人根本答不出。高级候选人会给出立体方案：工程上加 Schema Validation 拦截并把 Error 传回给模型做 Self-Correction；算法上通过调高 Temperature 采样或篡改实体来构造 Hard Negative，逼迫模型学习细粒度鉴别。
